# 🏏 Cricket Bat Product Page

This is a beginner-friendly frontend project that displays a product page for a cricket bat.
It is built using basic HTML and CSS to practice layout, styling, and UI components.

---

## 🚀 Features

* Product title and image display
* Price and stock information
* "Buy" and "Add to Cart" buttons
* Hover effects for better user interaction
* Simple and clean design

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📚 What I Learned

* Structuring a webpage using HTML
* Styling elements using CSS classes
* Adding hover and active effects
* Improving layout and alignment

---

## 🔧 Future Improvements

* Improve layout using Flexbox or Grid
* Add JavaScript for real functionality
* Enhance UI with better styling
